# hubot-oac

Scripts for our OaC-Guild

See [`src/oac.coffee`](src/oac.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-oac --save`

Then add **hubot-oac** to your `external-scripts.json`:

```json
[
  "hubot-oac"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
